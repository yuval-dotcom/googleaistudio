# נתוני ייבוא (Import Data)

העתק לכאן את קובץ האקסל לפני הרצת ה-seed.

**שם מומלץ:** `mangeassests.xlsx`  
(או כל שם – תעביר את הנתיב המלא ל-`npm run seed`)

**הרצת seed אחרי שהקובץ בתיקייה:**
```bash
npm run seed "data/mangeassests.xlsx" yuvalt12311@gmail.com
```

אם שינית את שם הקובץ, התאם את הנתיב בפקודה.
